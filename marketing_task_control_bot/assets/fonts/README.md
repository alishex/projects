# Shriftlar haqida

Loyiha mualliflik huquqi noma’lum font fayllarini o‘z ichiga olmaydi. Grafik servis avval Linux tizimidagi `DejaVuSans` yoki `LiberationSans` shriftlarini topishga harakat qiladi. Kali Linux va Ubuntu’da odatda DejaVu Sans mavjud bo‘ladi.

Zarur bo‘lsa, ochiq litsenziyali `.ttf` shriftni ushbu papkaga `DejaVuSans.ttf` va `DejaVuSans-Bold.ttf` nomlari bilan joylashtirish mumkin.
